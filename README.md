### 1. **Query Decomposition**

* **Prompt Example:**

### 2. **Decision-Making Prompts**

*Guide the model to choose among options or frameworks.*

* **Prompt Example (Report):**

  * *‚ÄúUse SWOT analysis (Strengths, Weaknesses, Opportunities, Threats) to evaluate electric vehicles in India.‚Äù*

### 3. **Semantic Filtering**

*Control style, tone, or relevance using constraints.*

* **Prompt Example (Article):**

  * *‚ÄúWrite a 500-word article on AI in healthcare, suitable for a newspaper audience, avoiding technical jargon.‚Äù*

* **Prompt Example (Case Study):**

  * *‚ÄúProvide a case study on Tesla‚Äôs marketing, but exclude financial data and focus only on customer engagement strategies.‚Äù*

### 4. **Role-based Prompting**

*Ask the model to adopt a perspective or role.*

* **Prompt Example (Report):**

  * *‚ÄúYou are a policy advisor writing a government report on cybersecurity threats.‚Äù*

*Encourage structured reasoning before the final output.*

* **Prompt Example (Article):**

  * *‚ÄúBefore writing the final article, first outline the key sections on AI in education, then expand each into full paragraphs.‚Äù*
